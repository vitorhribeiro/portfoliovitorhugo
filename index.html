Minhas Especialidades

HTML5

O HTML5 é a quinta e mais recente versão da linguagem de marcação usada para criar páginas web. Ele define a estrutura do conteúdo na web, como textos, imagens, vídeos, formulários, menus e muito mais.

CSS3

O CSS3 é a terceira versão da linguagem usada para estilizar páginas web. Ele define cores, layouts, fontes, efeitos visuais e animações, além de trabalhar com diferentes tamanhos de tela.

JAVASCRIPT

O JavaScript é uma linguagem de programação voltada para a interatividade e dinamismo de páginas web, permitindo respostas a ações do usuário e atualização de conteúdo em tempo real.

BOOTSTRAP

O Bootstrap é um framework front-end gratuito criado pelo Twitter. Ele acelera a criação de sites responsivos com componentes prontos de HTML, CSS e JS.

BANCO DE DADOS

Banco de Dados são sistemas que armazenam, organizam e gerenciam informações estruturadas. Essenciais para sites, sistemas de login e lojas virtuais.

PHP

O PHP é uma linguagem de programação backend que processa dados, conecta com bancos e gera páginas dinâmicas que o navegador do usuário recebe.
